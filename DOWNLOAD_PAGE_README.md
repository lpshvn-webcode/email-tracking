# 📥 Страница Скачивания (Ретро 2008 Стиль)

## 🎨 Что это?

Топорная страница в стиле 2008 года с кнопкой скачивания изображения. Когда получатель открывает ссылку из письма, он видит:

- 🔒 **Размытое изображение** с замком
- 🎯 **Огромную мигающую кнопку** "СКАЧАТЬ ФОТО"
- 🌈 **Градиенты и Comic Sans** как в старые добрые времена
- 📊 **Поддельные счетчики** просмотров и скачиваний
- 🔥 **Бегущие строки** с рекламными текстами

## 🚀 Как это работает:

### 1. Отправка письма

```bash
cd /Users/rabota/Desktop/email_backdating
python3 web_interface_advanced.py
```

Откройте http://127.0.0.1:5000 и отправьте письмо с изображением.

### 2. Получатель открывает письмо

В письме будет кнопка **"🖼️ Посмотреть фото"** которая ведет на страницу просмотра.

### 3. Страница просмотра

Получатель видит страшную страницу из 2008:

```
★★★ СУПЕР ФОТО ★★★

🔥🔥🔥 БЕСПЛАТНО!!! СКАЧАЙ СЕЙЧАС!!! 🔥🔥🔥

[Размытое изображение с замком 🔒]

⚠️ НАЖМИТЕ КНОПКУ ДЛЯ РАЗБЛОКИРОВКИ! ⚠️

┌─────────────────────────┐
│ ★ СКАЧАТЬ ФОТО ★        │
│ (100% БЕСПЛАТНО)        │
└─────────────────────────┘
```

### 4. При клике на кнопку:

✅ Изображение автоматически скачивается в папку Downloads  
✅ Записывается tracking: IP, User-Agent, время  
✅ Изображение разблокируется на странице  

## 📊 Что отслеживается:

### Три типа событий:

1. **`email_open`** - письмо открыто (tracking pixel)
2. **`image_view`** - страница просмотра загружена
3. **`download_click`** - кнопка скачивания нажата

### Данные в JSON:

```json

  "created_at": "2026-07-10T11:00:00",
```

## 🎯 Использование:

### Шаг 1: Запустите сервер
```bash

```

### Шаг 2: Отправьте письмо
- Откройте http://127.0.0.1:5000
- Загрузите изображение
- Заполните форму
- Отправьте

### Шаг 3: Получите tracking URL

- Видит ретро-страницу
- Нажимает кнопку
- Изображение скачивается
- Всё записывается!

## ⚠️ Важно:

### Для работы tracking нужен публичный доступ:

**Локально (только тесты):**
```
http://127.0.0.1:5000/view/ID
```

**С ngrok (для реальных получателей):**
```bash
ngrok http 5000
# Используйте: https://abc123.ngrok.io/view/ID
```

### Обновите URL в коде:
```python
# В web_interface_advanced.py, строка ~330:
image_url = f"https://abc123.ngrok.io/view/{tracking_id}"
tracking_pixel_url = f"https://abc123.ngrok.io/pixel/{tracking_id}.png"
```
