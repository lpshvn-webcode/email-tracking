# 🌐 Веб-Интерфейс для Отправки Email

## Удобный графический интерфейс для экспериментов

Вместо использования командной строки, вы можете использовать веб-интерфейс с красивой HTML формой для отправки писем с измененной датой.

---

## 🚀 Быстрый запуск

### 1. Убедитесь что зависимости установлены

```bash
cd /Users/rabota/Desktop/email_backdating
pip3 install -r requirements.txt
```

### 2. Запустите веб-сервер

```bash
python3 web_interface.py
```

Вы увидите:
```
============================================================
  WEB ИНТЕРФЕЙС ДЛЯ ОТПРАВКИ EMAIL С ИЗМЕНЕННОЙ ДАТОЙ
============================================================

✓ Сервер запущен на: http://127.0.0.1:5000
✓ Откройте браузер и перейдите по адресу выше

[Нажмите Ctrl+C для остановки]
```

### 3. Откройте браузер

Перейдите по адресу: **http://127.0.0.1:5000**

## 📋 Использование интерфейса

### Поля формы:

1. **Email отправителя** - адрес от кого письмо (может быть любым)
2. **Email получателя** - адрес куда отправить
3. **Тема письма** - заголовок письма
4. **Текст письма** - содержимое письма
5. **Дата и время** - дата которую увидит получатель в почтовом клиенте
6. **Вложение** - необязательно, можно прикрепить файл

### Процесс отправки:

1. Заполните все обязательные поля
2. При желании прикрепите файл (нажмите "Выберите файл")
3. Нажмите кнопку **"🚀 Отправить письмо"**
4. Дождитесь результата

### Результат:

**Успешная отправка:**
```
✅ Успешно отправлено!
Письмо успешно отправлено через mail.bientotmail.com

📅 Дата в письме: Tue, 07 Jul 2026 10:00:00 +0500
🕐 Реальное время: 10.07.2026 10:47:00

💡 Совет: Откройте полученное письмо и посмотрите исходный текст...
```

**Ошибка:**
```
❌ Ошибка отправки
[описание ошибки]

Возможные решения:
• Проверьте правильность email адресов
• Убедитесь что порт 25 не заблокирован
• Попробуйте использовать VPN
```

## 🔧 Настройка

### Изменение порта

По умолчанию сервер запускается на порту **5000**. Чтобы изменить:

Отредактируйте `web_interface.py`:
```python
app.run(debug=True, host='127.0.0.1', port=8080)  # Измените 5000 на 8080
```

### Доступ из сети

Чтобы открыть доступ с других устройств в сети:

```python
app.run(debug=True, host='0.0.0.0', port=5000)
```

Тогда можно будет открыть через IP компьютера, например:
```
http://192.168.1.100:5000
```

⚠️ **Внимание:** Не открывайте доступ в интернет без дополнительной защиты!

## 🛡️ Безопасность

### Важные замечания:

1. **Локальный доступ** - по умолчанию доступен только с вашего компьютера
2. **Нет аутентификации** - любой кто имеет доступ к интерфейсу может отправлять письма
3. **Временные файлы** - загруженные вложения автоматически удаляются после отправки
4. **Debug режим** - отключите для production использования

### Для production:

```python
# В web_interface.py измените:
app.run(debug=False, host='127.0.0.1', port=5000)
```

## 🐛 Решение проблем

### Ошибка "Address already in use"

Порт 5000 уже занят. Решения:
1. Закройте другое приложение использующее порт 5000
2. Измените порт в `web_interface.py`

### Ошибка "Template not found"

Убедитесь что существует папка `templates/` с файлом `index.html`:
```
email_backdating/
├── web_interface.py
└── templates/
    └── index.html
```

### Файл не прикрепляется

Проверьте:
1. Размер файла (максимум 16MB)
2. Права на запись в папку `uploads/`
