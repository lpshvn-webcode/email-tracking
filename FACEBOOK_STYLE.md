# 📧 Facebook Style Email Sender

Отправка писем в стиле Facebook с кнопкой "Confirm Email"

## 🎯 Что это?

Гибридное решение, которое использует:
- **Локальный сервер** - для отправки писем (порт 25)
- **Railway сервер** - для хостинга изображений и tracking

## ✨ Возможности

- ✅ Письма выглядят как настоящие от Facebook
- ✅ Синяя шапка с логотипом Facebook
- ✅ Кнопка "Confirm Email" синего цвета (#1877f2)
- ✅ При клике открывается ваше изображение
- ✅ Tracking: IP, User-Agent, время просмотра
- ✅ Backdating - поддельная дата отправки

## 🚀 Как использовать

### Шаг 1: Запустить локальный сервер

```bash
cd email_backdating
python3 web_interface_advanced.py
```

Откроется на: http://127.0.0.1:5000

### Шаг 2: Открыть Facebook страницу

Перейти по адресу:
```
http://127.0.0.1:5000/facebook
```

### Шаг 3: Загрузить изображение

1. Выбрать изображение
2. Нажать "Загрузить на Railway"
3. Скопировать полученную ссылку

**Важно:** Если Railway не настроен, изображение загрузится локально и ссылка будет `http://127.0.0.1:5000/view/...`

### Шаг 4: Заполнить форму

- **Кому**: email получателя
- **Имя получателя**: John, Sarah, etc.
- **Ссылка для кнопки**: вставить из Шага 3
- **От кого**: по умолчанию `security@facebookmail.com`
- **Дата и время**: поддельная дата

### Шаг 5: Отправить

Нажать "Отправить Facebook Email"

## 🔍 Tracking

После отправки можно отслеживать:

1. **Открытие письма** - когда получатель открыл
2. **Клик на кнопку** - когда нажал "Confirm Email"
3. **Просмотр изображения** - когда загрузилось фото
4. **IP адрес** - откуда открыли
5. **User-Agent** - устройство и браузер

Статистика доступна по адресу:
```
http://127.0.0.1:5000/tracking/[tracking-id]
```

или на Railway:
```
https://your-app.railway.app/tracking/[tracking-id]
```

## 🎨 Кастомизация

### Изменить отправителя

Можно использовать другие email-адреса Facebook:
- `security@facebookmail.com`
- `notification@facebookmail.com`
- `no-reply@facebookmail.com`

### Изменить шаблон

Редактировать файл: `templates/facebook_style_email.html`

Можно изменить:
- Цвета
- Текст
- Логотип
- Стиль кнопки

## 💡 Советы

1. **Используйте реальные имена** - письмо выглядит более убедительно
2. **Выбирайте правдоподобную дату** - не слишком старую
3. **Проверьте изображение** - перед отправкой откройте ссылку
4. **Tracking работает только с Railway** - локально tracking будет неполным

## ⚠️ Важно

- Отправка происходит **локально** через порт 25
- Railway **только для изображений** и страницы скачивания
- **Не требуется Gmail/SendGrid** для отправки
- Письма отправляются напрямую на MX сервер получателя

## 🐛 Проблемы

### Railway блокирует SMTP

**Решение:** Используется гибридный подход
- Railway = хостинг изображений ✅
- Локальный сервер = отправка писем ✅

### Порт 25 заблокирован

**Решение:** 
1. Использовать VPN
2. Связаться с провайдером
3. Использовать другую сеть

### Изображение не загружается

**Проверить:**
1. Railway сервер запущен
2. BASE_URL правильный в переменных окружения
3. Файл загрузился в папку `hosted_images/`

## 🚀 CLI версия

Можно отправить из командной строки:

```bash
python3 send_facebook_style.py \
  --to victim@example.com \
  --name John \
  --url "https://your-app.railway.app/view/abc-123" \
  --date "Mon, 1 Jan 2024 12:00:00 +0000"
```

## 🎯 Workflow

```
1. [Локально] Загружаете фото → Railway хостит
                ↓
2. [Локально] Получаете ссылку: https://railway.app/view/abc-123
                ↓
3. [Локально] Заполняете форму + вставляете ссылку
                ↓
4. [Локально] Отправляете письмо → Порт 25 → MX сервер
                ↓
5. [Получатель] Открывает письмо → видит Facebook design
                ↓
6. [Получатель] Кликает "Confirm Email" → открывается Railway
                ↓
7. [Railway] Показывает изображение + записывает tracking
                ↓
8. [Локально/Railway] Смотрите статистику
```
