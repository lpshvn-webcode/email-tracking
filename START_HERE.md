## 📦 Что это?

Комплексный инструмент для демонстрации уязвимости email протокола - возможности подделки заголовка `Date`. Проект включает:

✅ **CLI версию** - скрипт командной строки  
✅ **Web интерфейс** - красивая HTML форма  
✅ **Полную документацию** - для дипломной работы

## 🎓 Для дипломной работы

### Что демонстрирует проект:

1. **Техническая уязвимость**
   - Заголовок `Date` может быть подделан
   - Видимая дата ≠ реальная дата доставки

2. **Детекция подделки**
   - Заголовки `Received` содержат истину
   - Анализ полных заголовков письма

3. **Практическая реализация**
   - Два метода: CLI и Web
   - Прямая отправка на MX сервер
   - Поддержка вложений

## ❓ Частые вопросы

**Q: Письмо не приходит?**  
A: Проверьте папку СПАМ, подождите 1-2 минуты, проверьте что порт 25 не заблокирован.

**Q: Ошибка "порт 25 заблокирован"?**  
A: Используйте VPN или запустите на VPS сервере.

**Q: Можно ли отправить на Gmail/Yandex?**  
A: Да, укажите соответствующий email в получателе.

**Q: Безопасно ли это?**  
A: Проект для образовательных целей. Не используйте для незаконных действий.
